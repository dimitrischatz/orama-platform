Dashboard

A real-time summary of your CRM data at a glance.

The Dashboard is the default landing page when you open AccelCRM. It provides a high-level overview of your sales pipeline, contacts, and upcoming work.

Metric Cards

Four summary cards are displayed across the top of the dashboard:

Card What It Shows How It's Calculated
Total Contacts Number of people in your CRM Simple count of all contacts
Companies Number of organizations tracked Simple count of all companies
Pipeline Value Weighted value of open deals Sum of (deal value × probability) for all deals not in closed_won or closed_lost stages
Closed Won Total revenue from won deals Sum of the value of all deals in the closed_won stage

Open Deals Section

Below the metric cards, the dashboard lists up to 5 most recent open deals. Each entry shows:

  • Deal title
  • Current pipeline stage (as a colored badge)
  • Deal value in dollars
  • Win probability percentage

A "View all deals" link at the bottom navigates to the full Deals page.

Pending Activities Section

The dashboard also shows up to 5 pending (incomplete) activities. Each entry shows:

  • Activity title
  • Activity type (call, email, meeting, task, or note) as a color-coded badge
  • Due date, if one is set

A "View all activities" link at the bottom navigates to the full Activities page.

Tip: The dashboard updates in real-time as you add, edit, or remove contacts, companies, deals, and activities. There is no need to refresh.
