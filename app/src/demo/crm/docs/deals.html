Deals & Pipeline

Track sales opportunities from lead to close.

The Deals page is where you manage your sales pipeline. Every deal represents a sales opportunity with a dollar value and a probability of closing. Navigate to it via Deals in the sidebar, or by visiting /demo/crm/deals.

Page Layout

  • Page title "Deals" with a total count.
  • A view mode toggle: switch between Board (Kanban) and List (table) views.
  • An "Add Deal" button.

Deal Fields

Field Required Description
TitleYesName of the deal (e.g., "Enterprise License Renewal").
Value ($)YesDollar amount of the deal.
StageYesCurrent pipeline stage. See stages below.
ContactYesThe primary contact for this deal. Selected from existing contacts.
CompanyNoThe company associated with this deal. Selected from existing companies.
Probability (%)YesEstimated chance of winning, 0–100. Defaults to 50.
Expected Close DateNoTarget date for closing the deal.
NotesNoFree-text notes about the deal.

Pipeline Stages

Deals progress through six stages from left to right:

Stage Color Description Default Probability
LeadGrayA new, unqualified opportunity.10%
QualifiedBlueThe prospect's need and budget have been verified.30%
ProposalPurpleA proposal or quote has been sent.50%
NegotiationAmberTerms and pricing are being discussed.70%
Closed WonGreenThe deal has been successfully closed.100%
Closed LostRedThe deal was lost.0%
Auto-probability: When you drag a deal to a new stage on the Kanban board, the probability automatically updates to the stage's default value shown above. You can still override this manually by editing the deal.

Board View (Kanban)

The default view. Deals are displayed as cards organized into six columns, one per stage. Each column header shows the stage name, the number of deals in that stage, and the total dollar value.

Each deal card shows:

  • Deal title
  • Deal value (in green)
  • Contact name
  • Win probability
  • Edit and Delete buttons

Moving deals between stages

To move a deal to a different stage, drag the deal card from one column and drop it onto the target stage column. The deal's stage and probability will update automatically.

List View

Toggle to List view using the view mode button in the header. This displays deals in a table with the following columns:

Column Details
DealDeal title.
ValueDollar amount in green.
StageColored badge showing the current stage.
ContactName of the linked contact.
ProbabilityWin percentage.
ActionsEdit and Delete buttons.

Adding a Deal

  1. Click the "Add Deal" button.
  2. Fill in the Title, Value, and select a Contact (required).
  3. Choose the initial Stage, set the Probability, and optionally add a Company, Expected Close Date, and Notes.
  4. Click "Add Deal" to save.

Editing a Deal

  1. Click the pencil icon on any deal card (Board view) or row (List view).
  2. Update any fields in the modal.
  3. Click "Update Deal" to save changes.

Deleting a Deal

  1. Click the trash icon on the deal card or row.
  2. Confirm the deletion in the dialog.
