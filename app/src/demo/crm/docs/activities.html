Activities

Log calls, emails, meetings, tasks, and notes.

The Activities page is your task and interaction log. Use it to track every touchpoint with your contacts and deals. Navigate to it via Activities in the sidebar, or by visiting /demo/crm/activities.

Page Layout

  • Page title "Activities" with a total count.
  • Filter chips: All, Pending, Completed.
  • An "Add Activity" button.

Activities are displayed as a vertical list of cards.

Activity Fields

Field Required Description
TypeYesOne of: call, email, meeting, task, or note.
TitleYesShort description of the activity.
DescriptionNoLonger details (textarea).
ContactNoLink this activity to a specific contact.
DealNoLink this activity to a specific deal.
Due DateNoWhen this activity is due.

Activity Types

Each type has a distinct color and icon for quick visual identification:

Type Color Icon Typical Use
callBluePhonePhone calls with contacts.
emailPurpleEnvelopeEmails sent or to send.
meetingGreenCalendarScheduled meetings or demos.
taskAmberCheckboxInternal to-do items (e.g., "Prepare contract").
noteGrayPencilGeneral notes or observations.

Activity Cards

Each activity card contains:

  • A checkbox on the left to toggle completion.
  • A color-coded icon matching the activity type.
  • The activity title (with strikethrough styling if completed).
  • The description, if present.
  • Linked contact name (with a person icon), if set.
  • Linked deal name (with a currency icon), if set.
  • The due date (with a calendar icon), if set.
  • A type badge on the right.
  • Edit and Delete buttons (visible on hover).

Filtering Activities

Use the filter chips at the top of the page:

  • All — shows every activity regardless of status.
  • Pending — shows only incomplete activities.
  • Completed — shows only finished activities.

Marking an Activity Complete

Click the checkbox on the left side of any activity card. The activity will:

  • Show an animated checkmark.
  • Have its title displayed with a strikethrough style and reduced opacity.
  • Move to the "Completed" filter group.

Click the checkbox again to mark it as pending.

Adding an Activity

  1. Click the "Add Activity" button.
  2. Select the activity Type from the dropdown.
  3. Enter a Title (required).
  4. Optionally add a Description, link a Contact, link a Deal, and set a Due Date.
  5. Click "Add Activity" to save.

New activities always start as pending (not completed).

Editing an Activity

  1. Hover over the activity card to reveal the action buttons.
  2. Click the Edit button.
  3. Update any fields in the modal.
  4. Click "Update Activity" to save.

Deleting an Activity

  1. Hover over the activity card to reveal the action buttons.
  2. Click the Delete button.
  3. Confirm the deletion in the dialog.
